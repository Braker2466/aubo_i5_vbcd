# AUBO-i5 机械臂视觉控制项目

## 1. 项目简介
本项目面向 AUBO-i5 机械臂控制与深度相机配合应用，包含机械臂 SDK 封装、相机接口、手眼/工具标定、点位管理、控制示例与数据采集脚本。
同时包含了linux/windows的官方库文件。
项目的核心目标不是从零重写厂商 SDK，而是在现有控制链路上进行可控的优化与扩展。

## 2. 运行环境
本项目使用现有 conda 环境运行。

基本原则：
- 使用已有 conda 环境
- 不创建新的虚拟环境
- 不随意升级核心依赖
- 真机运行前先确认动态库、设备连接、标定文件均有效

启动前示例：

```bash
conda activate <你的环境名>
cd <项目根目录>
