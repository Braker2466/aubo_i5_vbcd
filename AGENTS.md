# AGENTS.md

## 项目概述
本仓库是一个真实机械臂控制项目，控制对象为 AUBO-i5 机械臂，并结合深度相机完成视觉相关任务。
本项目不是纯算法 demo，也不是普通 Python 教学工程。安全性、接口稳定性、标定一致性、运行环境一致性，优先级高于激进重构。

Codex 在本仓库中的默认职责是：
- 理解现有代码结构
- 做小步、可回退的优化与扩展
- 优先增强日志、文档、测试、dry-run、回放能力
- 谨慎处理真实硬件控制链路

## 环境约束
- 使用当前已经激活的 conda 环境。
- 不要创建新的虚拟环境。
- 不要主动升级 Python 版本。
- 不要主动升级核心依赖。
- 任何 pip install / conda install / apt install 操作，必须先说明将安装什么、为什么需要，再等待确认。
- 除非被明确要求，不要修改系统级环境变量、驱动路径、动态库搜索路径。

## 真机安全规则
- 默认假定当前项目连接的是真实 AUBO-i5 机械臂，而不是仿真对象。
- 不要自动运行任何可能导致机械臂运动的脚本。
- 不要自动执行 connect、startup、power on、move、io trigger、夹爪/执行器动作等操作。
- 涉及真实硬件动作的修改，先说明改动点、潜在风险、验证方式，再实施。
- 优先实现 dry-run、mock、日志回放、离线验证，而不是直接改真机执行路径。
- 不要删除已有异常处理、限位保护、人工确认逻辑、速度限制逻辑。
- 不要默认提高速度、加速度、插补频率或执行器触发频率。

## 坐标系、单位、姿态表示
本项目中以下约定属于高敏感内容，未经明确要求不得更改：
- 位置单位
- 角度单位
- RPY 顺序或四元数顺序
- 基座坐标系、工具坐标系、相机坐标系定义

2. 列出差异；
3. 给出最小改动建议；
4. 等确认后再修改。
